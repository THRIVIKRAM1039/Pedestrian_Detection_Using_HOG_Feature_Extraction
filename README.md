# Pedestrian Detection Using HOG Feature Extraction

## Overview
This project detects pedestrians using HOG (Histogram of Oriented Gradients) feature extraction and OpenCV in Python.

## Technologies Used
- Python
- OpenCV
- NumPy
- Computer Vision

## Features
- Detects pedestrians from images/videos
- Uses HOG descriptor method
- Real-time object detection
- Bounding box visualization

## Project Working
The system processes image frames using OpenCV and extracts HOG features to identify pedestrians.

## How to Run

1. Install Python
2. Install required libraries

```bash
pip install opencv-python numpy
```

3. Run the Python file

```bash
python app.py
```

## Output
The project highlights detected pedestrians using rectangles.
